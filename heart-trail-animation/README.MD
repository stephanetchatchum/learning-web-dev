# Heart Trail Animation

A simple mouse-tracking animation that leaves colorful floating hearts behind the cursor.

## Overview

This small web project uses plain HTML, CSS, and JavaScript to create a heart-shaped trail effect. When you move the mouse over the page, heart icons are generated at the pointer position and animate upward while fading out.

## Files

- `index.html` - The page structure and script inclusion.
- `style.css` - Page styling and heart animation keyframes.
- `script.js` - Mouse movement tracking and dynamic heart creation.

## How to Run

1. Open `heart-trail-animation/index.html` in a browser.
2. Move the mouse around the page.
3. Hearts will appear and float up before disappearing.

## Features

- Mouse movement event handling
- Dynamic DOM element creation
- Animated heart trail using CSS keyframes
- Clean fullscreen black background for contrast

## Notes

- The heart image is loaded from an external URL.
- Each heart element is removed after 3 seconds to keep the page lightweight.
- The heart size is randomized for a playful effect.

Enjoy the animation!
